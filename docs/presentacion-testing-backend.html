Construir confianza repetible, no solo encontrar errores tarde

RealistaAPI eCommerce con pedidos, stock, pagos y endpoints HTTP.
ProfesionalIncluye CI/CD, cobertura, mocks, integracion y performance.

El curso ensena a decidir que probar, en que nivel y con que costo de mantenimiento.

Arquitectura

Separacion por capas para probar sin ruido

Domain

Entidades, estados, excepciones y reglas de negocio.

Application

Casos de uso, DTOs e interfaces de repositorios/gateways.

Infrastructure

Repositorios in-memory y gateway de pago simulado.

API

Controllers, DI, health checks y traduccion HTTP.

La regla importante: el negocio se prueba cerca de Domain, y HTTP se prueba con WebApplicationFactory.

Dominio eCommerce

Un backend pequeno, pero con reglas reales

Entidades

  • Product, Customer, Order, OrderItem, Payment.
  • OrderStatus y PaymentStatus.
  • Excepciones de dominio para reglas incumplidas.

Casos de uso

  • Crear, consultar y cancelar pedidos.
  • Calcular total y validar stock.
  • Simular pagos aprobados o rechazados.

Estrategia

Piramide de pruebas aplicada al curso

03 Integration Tests

Integracion: la API completa en memoria

  • `WebApplicationFactory<Program>` levanta la API sin puertos.
  • Se ejercitan routing, model binding, controllers, DI y servicios reales.
  • `POST /api/orders` devuelve `201 Created` cuando el pedido es valido.
  • La falta de stock llega a HTTP como `400 Bad Request`.

04 Smoke Tests

Smoke tests: una alarma temprana

ArranqueLa aplicacion construye el host correctamente.
Salud`GET /health` responde OK.
Endpoint critico`GET /api/products` esta disponible.

No sustituyen pruebas profundas. Sirven para detectar fallas obvias antes de avanzar.

05 End-to-End Tests

Buenas practicas

Como escribir pruebas que sobreviven al proyecto

  • Nombre con escenario y resultado esperado.
  • Arrange, Act, Assert evidente.
  • Datos pequenos y expresivos.
  • Un motivo de fallo por prueba.
  • Evitar sleeps y aleatoriedad innecesaria.
  • Usar builders o fixtures cuando los datos crezcan.
