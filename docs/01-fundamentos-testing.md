# Fundamentos de testing

Testing profesional busca confianza repetible, no solo encontrar errores al final.

Conceptos base:

- Unit test: valida una regla pequena sin infraestructura externa.
- Mocking test: reemplaza colaboradores para verificar decisiones de aplicacion.
- Integration test: prueba componentes reales conectados en memoria o en un entorno controlado.
- Smoke test: confirma que el sistema arranca y responde lo minimo indispensable.
- End-to-end test: recorre un flujo de usuario o negocio completo.
- Performance test: mide comportamiento bajo carga.

En este curso la piramide se aplica de forma practica: muchas pruebas de dominio, algunas de integracion y pocas pruebas end-to-end.
